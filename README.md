# Portal Financeiro Casas Bahia

Bem-vindo ao Portal Financeiro da Casas Bahia! Este projeto é uma plataforma web moderna e intuitiva desenvolvida para auxiliar os clientes da Casas Bahia a gerenciar suas compras, negociar acordos de forma flexível, processar pagamentos e acessar uma gama de produtos financeiros exclusivos.

## 🚀 Principais Funcionalidades

-   **Validação e Status do CPF**: Verificação instantânea do status financeiro do cliente via CPF.
-   **Simulador de Acordos**: Ferramenta interativa para simulação e negociação de dívidas, oferecendo planos de pagamento personalizados e descontos.
-   **Processamento de Pagamentos**: Suporte a diversas formas de pagamento, incluindo PIX, boleto bancário e cartão de crédito, com confirmação e histórico.
-   **Produtos Financeiros**: Acesso fácil a produtos como conta digital, cartão de crédito Casas Bahia, empréstimos pessoais e seguros.
-   **Dashboard Personalizado**: Área do cliente com visão geral de contratos, próximos vencimentos, valor total e status da conta.
-   **Adiantamento de Parcelas**: Opção para adiantar pagamentos e obter descontos sobre juros.
-   **Assistente Virtual (CBIA)**: Um assistente inteligente para responder dúvidas e guiar o usuário na plataforma.
-   **Responsividade Total**: Design adaptável para proporcionar a melhor experiência em qualquer dispositivo (desktop, tablet, mobile).

## 💻 Tecnologias Utilizadas

Este projeto é construído com as seguintes tecnologias e bibliotecas:

-   **Frontend**:
    -   **React 18.2.0**: Biblioteca JavaScript para construção de interfaces de usuário.
    -   **Vite**: Ferramenta de build rápida e eficiente para desenvolvimento frontend.
    -   **TailwindCSS 3.3.2**: Framework CSS utility-first para estilização rápida e responsiva.
    -   **Framer Motion 10.16.4**: Biblioteca para animações e transições fluidas na interface.
    -   **shadcn/ui**: Componentes de UI construídos com [Radix UI](https://www.radix-ui.com/) para acessibilidade e personalização.
    -   **Lucide React 0.292.0**: Coleção de ícones vetoriais.
    -   **React Router 6.16.0**: Para gerenciamento de rotas e navegação.
-   **Backend & Armazenamento (Recomendado para Produção)**:
    -   **Supabase**: Para autenticação, banco de dados e armazenamento persistente. (Atualmente, a implementação inicial usa `localStorage` para prototipagem rápida).
-   **Outros**:
    -   **JavaScript**: Linguagem de programação principal.
    -   **Native Fetch API**: Para requisições HTTP.
    -   **React Helmet**: Para gerenciamento de metadados do documento (SEO).
